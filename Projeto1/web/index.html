História da web

A história da internet se dá desde a década de 60 quando foi idealizado o primeiro conceito de internet para os aliados na guerra se comunicarem. Juntamente com ela, vieram ideias de criptografia, interceptação de mensagens, etc. Após isso, nos EUA, essa ideia continuou forte em instituições de pesquisa, militares e educacionais, onde eles conseguiram criar uma rede que conectava algumas principais universidades com centros de pesquisa, instalações militares e o Pentágono, essa rede era chamada de Arpanet. Com o tempo, essa rede se aprimorou, criando protocolos de comunicação, se tornando mais segura. Por volta de 1980, a uso militar da rede foi migrado para a Milnet, uma rede de uso exclusivamente militar. Assim a Arpanet passou a possuir uso científico e comercial. Por volta de 1990 houve a criação do WWW, sistema de distribuição de documentos e informação, e foi através dele que surgiram os primeiros sites. Com o tempo foram criados os navegadores que se aprimoraram até a tecnologia que possuímos hoje. Atualmente existem milhões de sites com várias utilidades diferentes, redes sociais, várias tecnologias para desenvolvimento web, etc.

O que são Clients

Clients são, em uma comunicação dispositivo-servidor, que ocorre em uma rede, o dispositivo que está acessando sites e recebendo dados dele. Pode se tratar e um desktop, um notebool, um celular, qualquer dispositivo que acessa a rede e faz requisições nela para informações através de sites ou aplicativos.

O que são Servers

Já os Servers estão do outro lado da conexão existente na rede. São esses os servidores que estão hospedando esses sites, são neles que as informações que são requisitadas pelos Clients estão. A comunicação se dá de forma que um dispositivo qualquer acessa alguma página da internet, faz uma requisição de informações, e essa requisição viaja pela rede até o servidor onde essas informações estão. Após isso, o servidor responde essa requisição com as informações que foram pedidas.

Linguagens de programação

As linguagens de programação geralmente utilizadas para o lado client são HTML, CSS e Javascrit, que são as linguagens básicas para desenvolvimentos de sites. Após o aprendizado dessas linguagens, é recomendado o aprendizado de um framework para javascript, como React, Angular, etc.

Do lado Server estão linguagens de programação voltadas ao back-end como Python, C#, C++, Ruby, Java.

Ferramentas utilizadas

A ferramenta a ser utilizada será principalmente o Visual Studio Code, a IDE responsável pelo desenvolvimentos dos sites. Um plugin utilizado dentro dela é o Live Server, o qual atualiza o site referente a página sendo desenvolvida cada vez que ela é atualizada na IDE.

Tags do HTML

A linguagem HTML é baseada inteiramente em tags, que são marcadores que irão posicionar o texto na página de acordo com a forma de desenvolvimento. Existem tags para título que vão de h1 a h6, tags de parágrafo como p, de link como a, de realce de texto como strong, i, u, de listas como ul, ol, li, entre várias outras tags que nos ajudam a montar o texto das páginas de maneira que ele fique o mais bem posicionado e legível possível.
